You can defined a global prop here that is passed into every component.

```json
{
  "name": "your site",
  "version": "1.0.0",
  "description": "",
  "cuttlebelle": {
    "site": {
      "globalProp": {
        "fo": "bar"
      }
    }
  }
}
```
